## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Joshuakibwage/code-tutor-landingpage.git
   cd code-tutor-landingpage
   ```

2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```
